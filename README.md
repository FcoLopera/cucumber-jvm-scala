# cucumber-jvm-scala
Cucumber Scala

## Dependency

### Maven
To use cucumber-jvm-scala in your project, add the following dependency to your `pom.xml`:


```xml
<dependency>
    <groupId>io.cucumber</groupId>
    <artifactId>cucumber-scala</artifactId>
    <version>{{% version "scala" %}}</version>
    <scope>test</scope>
</dependency>
```
